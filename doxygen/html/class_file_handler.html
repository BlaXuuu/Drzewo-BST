Opis szczegółowy

Klasa obslugujaca operacje plikowe dla drzewa BST.

  • Umozliwia zapisywanie stanu drzewa do plikow tekstowych i binarnych oraz odczytywanie danych (zastepujac lub dodajac do) drzewa. Jest zaprzyjazniona z klasa BST, aby miec dostep do jej prywatnych metod.
